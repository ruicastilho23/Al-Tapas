## 🚀 Quick Start Guide:

### Option 1: GitHub Pages (FREE Hosting)
1. Go to GitHub.com → Sign in/Create account
2. Click "New repository"
3. Name: `al-tapas-modern` (make it Public)
4. Upload all 4 files from this ZIP
5. Go to Settings → Pages → Select "main" branch
6. Your site will be live at: `https://yourusername.github.io/al-tapas-modern/`

### Option 2: Local Testing
1. Extract all files to a folder
2. Double-click `index.html` to open in browser
3. All functionality works locally

## 🔧 Customization:

### Edit Menu Items:
- Open `index.html` in any text editor
- Find sections marked with `<!-- EDITABLE: -->`
- Update menu items, prices, contact info

### Change Colors:
- Open `style.css`
- Modify the CSS variables at the top:
  ```css
  --color-primary: #C41E3A;          /* Spanish Red */
  --gradient-hero: linear-gradient(135deg, #8B0000 0%, #2d1b1b 50%, #1a1a1a 100%);
  ```

### Add Favicon (Optional):
- Create a 16x16 pixel .ico file
- Name it `favicon.ico`
- Place in same folder as `index.html`
